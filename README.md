# Pasit P. Portfolio

A modern, dark-themed portfolio for a Data Analyst, built with HTML and CSS using the JetBrains Mono font.

## Features
- **Modern Dark Theme**: Inspired by developer-centric aesthetics.
- **Responsive Design**: Works on desktops, tablets, and mobile devices.
- **Performance Focused**: Minimal dependencies and fast loading times.
- **Typography**: Uses JetBrains Mono for a precise, data-focused feel.

## Deployment to GitHub Pages

To host this site on GitHub Pages, follow these steps:

1. **Create a GitHub Repository**:
   - Go to GitHub and create a new repository (e.g., `my-portfolio`).
   - Keep it public.

2. **Upload your files**:
   - Upload `index.html`, `style.css`, and any other assets (like images) to the repository.

3. **Enable GitHub Pages**:
   - Go to the **Settings** tab of your repository.
   - Click on **Pages** in the left sidebar.
   - Under "Build and deployment", ensure "Deploy from a branch" is selected.
   - Select the `main` branch and the `/ (root)` folder, then click **Save**.

4. **View your site**:
   - GitHub will give you a URL (e.g., `https://yourusername.github.io/my-portfolio/`).
   - It may take a minute or two for the site to go live.

## Customization
- **Projects**: Edit the `index.html` file to update project descriptions or links.
- **Contact**: Update the links in the footer section of `index.html` with your actual LinkedIn and GitHub profiles.

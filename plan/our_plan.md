# Data Analyst Portfolio Plan: Pasit P.

This plan outlines the creation of a professional, modern data analyst portfolio for Pasit P., designed for deployment on GitHub Pages.

## 1. Visual Design & Aesthetic
*   **Theme**: Modern "Developer-centric" Dark Mode.
    *   **Background**: Deep Matte Black (`#0d1117`).
    *   **Surface**: Dark Gray (`#161b22`) for project cards.
    *   **Accents**: Cyan or Neon Blue (`#58a6ff`) for links and highlights.
*   **Typography**: **JetBrains Mono** (imported via Google Fonts) to give it a precise, data-focused feel.
*   **Aesthetics**: Minimalist borders, subtle transitions, and high-readability contrast.

## 2. Technical Architecture
*   **Structure**: Semantic HTML5 for SEO and accessibility.
*   **Styling**: **Vanilla CSS3** (as per project standards) using CSS Variables for easy theme adjustments and CSS Grid/Flexbox for a responsive layout.
*   **Assets**: SVG icons for social links to ensure sharpness at any resolution.

## 3. Content Structure
*   **Hero Section**:
    *   **Name**: Pasit P.
    *   **Role**: Data Analyst
    *   **Tagline**: Transforming complex data into actionable insights through precision and clarity.

*   **Project Grid (3 Featured Projects)**:
    1.  **Sales Performance Dashboard**: Interactive visualization using Python and SQL to identify growth opportunities.
    2.  **Customer Segmentation Analysis**: Machine learning approach (Scikit-learn) to categorize customer behaviors.
    3.  **Supply Chain Optimization**: Analytical model built in Excel/PowerBI to reduce operational costs.

*   **Contact & Socials**:
    *   LinkedIn Profile
    *   GitHub Profile
    *   Email Address

## 4. Step-by-Step Implementation Roadmap
1.  **Setup**: Create `index.html` and `style.css` in the root directory.
2.  **Typography & Variables**: Import JetBrains Mono and define the color palette in CSS.
3.  **Hero Implementation**: Build the main intro section with a focus on typography.
4.  **Project Grid**: Develop the responsive grid layout for the three featured projects.
5.  **Contact & Socials**: Add a clean footer with functional icons.
6.  **Responsive Refinement**: Ensure the site looks excellent on both 4K monitors and mobile devices.
7.  **Deployment Prep**: Finalize the structure for GitHub Pages (ensuring all paths are relative).
